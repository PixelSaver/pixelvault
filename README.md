## About The Project

### Abstract
PixelVault is a secure locally encrypted gui password manager written with rust and egui.
Features include
* Vault encryption
* Service, Username, Password entries
* Searching usernames and services


### What do I take out of this?
First rust project! Rust docs are a joy!
